# hpiv1-stop-codon-analysis
Scripts used for analyses in the associated manuscript.
## Overview
This repository contains scripts used for the analyses described in the
associated manuscript.

## Analyses and scripts
- Sequence selection and definition of the pseudo-V reading frame in HPIV-1  
  (R scripts)

- Stop codon counting in pseudo-translated reading frames of the N, P, M, F,
  and HN genes of HPIV-1 and SeV  
  (R scripts)

- Forward simulation of overlapping reading frames  
  (Python scripts)

## Notes
File paths and input data should be adapted to the local environment.
